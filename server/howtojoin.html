How to Join the Server

First, ask Tom to add you to the server whitelist. He will make sure you aren't a griefer or a troll.

You can apply to join here.

Tom will also give you the server port as an extra layer of security for the server.

For Java Edition

  1. Launch Minecraft Java Edition.
  2. Click on Multiplayer from the main menu.
  3. Click Add Server.
  4. Enter the server name and address:
    • Server Address: mc.tommustbe12.com:port
  5. Click Done and select the server from the list to join.

For Bedrock Edition (PC or Mobile)

  1. Launch Minecraft Bedrock Edition.
  2. Click on Play and go to the Servers tab.
  3. Scroll down and click Add Server.
  4. Enter the server details:
    • Server Name: Your Server Name
    • Server Address: mc.tommustbe12.com
    • Port: (port given by TomMustBe12)
  5. Click Save and select the server to join.

For Consoles (Xbox, PlayStation, Switch)

  1. Friend 'BCMain1'. Then, join the world they are hosting in the Friends tab.
  2. Click on 'Connect to server'.
  3. Enter these server details:
    • Server Address: mc.tommustbe12.com
    • Server Port: (port given by TomMustBe12)
    • Put whatever display name you prefer (Recommended: Ohio Sigmas SMP).
    • Make sure to click "Add to server list" in order to not have to type in server details again (you'll just need to click on the server list item that corresponds to the server).
    • From now on, join the world hosted by "BCMain1" and always join the server with the server details you provided.
  4. Follow the app's instructions to connect your console to the server.

If you encounter any issues, feel free to contact support or check the server's documentation for troubleshooting tips.
