HTML Basics

HTML is the structure of web pages. Here's an example of a simple HTML snippet:

<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

CSS Basics

CSS is used for styling HTML elements. Example:

body {
  background-color: black;
  color: white;
  font-family: 'BerkeleyMono', monospace;
}

JavaScript Basics

JavaScript makes web pages interactive. Example:

document.addEventListener('DOMContentLoaded', () => {
  alert('Welcome to Learn to Code!');
});
