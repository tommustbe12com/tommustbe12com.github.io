Server Rules

Please follow these server rules so you don't get banned from the TomMustBe12 Public SMP Server:


  1. You have to listen to what TomMustBe12 or any [upcoming] admins will say.
  2. No spawn killing
  3. No spawn trapping or portal trapping
  4. No bad words
  5. No innappropriate content
  6. Griefing is allowed, but to a certain extent. Do not continuously grief or do it exceedingly.
  7. Don't target anybody.
  8. Have fun!

Keep in mind that if you do not follow these rules, you may get banned from the server for a long time. You will get a warning (a kick) the first time you are caught disobeying the rules. The second time, you will be given a more severe punishment.
