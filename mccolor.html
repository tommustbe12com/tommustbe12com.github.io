Color Picker for Minecraft

In Minecraft, especially servers like Hypixel, there are colors and stuff you can use. So this is here to give you a guide on what colors do what.

Each color will be listed below.



&0 - Black

&1 - Dark Blue

&2 - Dark Green

&3 - Dark Aqua

&4 - Dark Red

&5 - Dark Purple

&6 - Gold

&7 - Gray

&8 - Dark Gray

&9 - Blue

&a - Green

&b - Aqua

&c - Red

&d - Pink

&e - Yellow

&f - White

&g - Mustard


Format Codes

These are used with colors, and have similar syntax. However, it changes how the text displays.

Each format code will be listed below.


&u - Underline

&l - Bold

&o - Italic

&m - Strikethrough

&k - Scrambled

&r - Reset *




* Reset leaves the consequent text after the reset to be clear of all the previous color codes and format codes used.
